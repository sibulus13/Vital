## Short Description (for listing/search)

## Long Description (COPY THIS INTO DESCRIPTION TEXTBOX)

Vital is a sleek, minimal watchface designed for the Fitbit Versa 2. It displays essential information with a focus on readability and simplicity. The watchface features a black background with carefully chosen typography and color accents for optimal visibility and battery efficiency.

Key Features:

- Large, bold digital time display for easy reading
- Current date in abbreviated format
- Real-time heart rate monitoring
- Color-coded battery indicator
- Clean, minimalist design with high contrast
- Always-on display support

Perfect for users who want essential information at a glance without visual clutter.
